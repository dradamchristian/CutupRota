# Cut-Up Rota Booking (V1)

Vite + Vanilla JS + Supabase + Netlify Functions implementation for pathology bench booking.

## Environment variables

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `ADMIN_PIN`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
